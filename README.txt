Codevelp CanvasVirtual
----------------------

Este proyecto contiene manifiestos html, css y Js
para que unificados se pueda dibujar con la API de
canvas de una manera mas rápida y virtual utilizando el
mouse para diseñar cualquier estructura compleja y luego 
simplemente copiar el código generado por el programa,
la verdad es muy útil como herramienta muy simple que es
Codevelp CanvasVirtual.
